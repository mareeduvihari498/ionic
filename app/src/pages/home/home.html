Welcome to the Ionic Example App. Here you can see different examples on how to use the Ionic Framework.

About the app

These examples are made in my spare time and the way I think they could be implemented. My way is not perfect and I am always willing to learn. So if you have any remarks or improvements, don't hesitate to create an issue on the github page.

Api and data storage

With some cordova plugins it is required to connect the app to an external resource, for example the upload part of the transfer plugin. For that I will create an API (written in the C# dotnet core framework). This API will run on the following endpoint https://api.dtaalbers.com/ionic-example-app. This api will recieve and store and data you send to it. So be aware of that. Docs will be available when it goes live.
